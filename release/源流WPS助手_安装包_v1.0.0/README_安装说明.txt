源流WPS助手 安装说明
版本：v1.0.0
出品：源流数字

一、安装步骤
1. 解压本安装包到任意目录（路径建议不含特殊字符）
2. 双击 install.bat
3. 按提示保存并关闭 WPS 文档
4. 输入 Y 确认继续安装
5. 安装程序会自动关闭 WPS 并完成安装
6. 看到“安装成功”提示后，重新打开 WPS 文字
7. 顶部出现「源流WPS助手」即安装成功

说明：每次安装会生成新的安装标识；卸载后重新安装将重新显示首次使用引导页，但不会清除您已保存的 DeepSeek API Key。

二、使用说明
1. 点击顶部「源流WPS助手」选项卡
2. 点击「打开助手」
3. 首次使用可免费试用 3 天
4. 购买激活码后可点击“我有激活码，立即激活”
5. 使用前需填写您自己的 DeepSeek API Key

三、联网说明
- 插件界面本地运行，不依赖本地开发服务器
- 激活验证需要联网（api.sourceflower.com）
- DeepSeek 调用需要联网
- 用户需使用自己的 DeepSeek API Key

四、卸载方式
1. 双击 uninstall.bat
2. 按提示保存并关闭 WPS 文档
3. 输入 Y 确认继续卸载
4. 安装程序会自动关闭 WPS 并完成卸载
5. 看到“卸载完成”提示后，重新打开 WPS 确认插件已移除

说明：卸载仅删除本插件目录与配置，不会清理 WPS 公共缓存或其他加载项数据。

五、常见问题
- 看不到「源流WPS助手」：请完全退出 WPS 后重新打开（确保任务栏无 WPS 图标）
- 安装失败：请右键 install.bat，以管理员身份运行
- 卸载后仍有残留：联系源流数字技术支持
- 仍失败：联系源流数字技术支持

技术支持：源流数字
