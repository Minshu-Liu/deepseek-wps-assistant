# 源流 WPS智能助手 BYOK版安装说明

**版本：v1.0.6 BYOK 纯净版**  
**适用对象：普通用户（无需编程基础）**

---

## 1. 这是什么

源流 WPS智能助手 BYOK版（Sourceflower WPS Assistant）是一款安装在 **WPS 文字** 里的文档 AI 辅助工具，可以帮助您：

- 润色、总结、纠错、扩写、改写文档内容  
- 以及续写、缩写、摘要、中英翻译等常用能力（以插件界面为准）

**重要说明：**

- 使用前需要您 **自己准备 DeepSeek API Key**（类似一把“钥匙”），在插件里填写并保存后才能使用。  
- 本插件 **不包含** DeepSeek 模型额度，也 **不会** 替您充值或代扣 API 费用。  
- 调用 AI 时产生的费用，由 **DeepSeek 官方** 按您的账号规则收取，请您自行关注余额。  
- AI 生成的内容 **仅供参考**，重要材料请务必人工核对。

您购买的 **¥29.9** 对应的是：**安装包与基础使用支持**（见本文第 10 节），不包含 DeepSeek 账号里的 API 费用。

## 2. 使用前准备

请确认您已具备以下条件：

1. **Windows 电脑**（与当前安装包一致）  
2. 已安装 **WPS Office**（能正常打开 WPS 文字）  
3. 已获取 **DeepSeek API Key**（在 DeepSeek 官方平台申请）  
4. 已下载 **源流 WPS智能助手安装包**（zip 压缩包，请从您购买或官方提供的渠道获取）  
5. 建议先 **保存文档并关闭 WPS**（安装、卸载时都需要）

---

## 3. 安装步骤

请按顺序操作：

1. **解压** 安装包到任意文件夹（例如“桌面\源流WPS助手”）。  
   - **不要** 直接在压缩包里双击安装程序。  
2. 打开解压后的文件夹，找到 **`install.bat`**。  
3. **双击运行** `install.bat`。  
   - 若 Windows 弹出安全提示，请确认文件来自 **您信任的官方渠道** 后再选择允许运行。  
4. 按黑色窗口里的中文提示操作：先保存并关闭 WPS 文档。  
5. 当提示是否继续安装时，输入 **`Y`**（大写或小写均可），然后按 **Enter（回车）**。  
6. 等待窗口提示 **安装成功**（程序可能会自动帮您关闭 WPS 进程）。  
7. **重新打开 WPS 文字**。  
8. 看顶部是否出现 **「源流WPS助手」** 选项卡或入口。  
   - 能看到即表示安装基本完成，下一步请配置 API Key。

---

## 4. 配置 DeepSeek API Key

1. 在 WPS 中打开 **源流 WPS智能助手**（点击「打开助手」等入口）。  
2. 进入 **设置** 区域。  
3. 在 **DeepSeek API Key** 输入框中，粘贴您自己的 Key。  
4. 点击 **保存**。  
5. 看到保存成功提示后，即可使用润色、总结、纠错等功能。  

**安全提醒：**

- API Key 相当于您的账号钥匙，**不要发给他人**，也不要发到群聊、朋友圈截图里。  
- Key 一般只保存在 **您本机** 的插件环境中。

**费用说明：**

- DeepSeek API 的调用费用由 **DeepSeek 官方** 按您的账号计费。  
- **本插件不收取** API 调用费。  
- **本插件不提供** 免费模型额度或内置额度。

**获取 Key：** 请登录 DeepSeek 官方开放平台，按页面指引创建 API Key（具体页面以 DeepSeek 官方为准）。

## 5. 基本使用方法

典型流程如下：

1. 在 WPS 文档里 **选中** 需要处理的文字（也可以不选中，单独在下方输入问题）。  
2. 打开 **源流 WPS智能助手**。  
3. 点击需要的功能，例如 **润色、总结、纠错、扩写、改写** 等。  
4. 如需控制回答长短，可在 **「回复长度：控制在 ___ 字以内」** 处填写字数（见下一节）。  
5. 等待 AI 生成结果。  
6. 查看生成内容是否满意。  
7. 若满意，可使用 **「替换选中文字」** 写回文档；也可以自行复制粘贴到需要的位置。

**小提示：** 使用快捷功能前，通常需要先 **配置并保存** DeepSeek API Key。

---

## 6. 回复长度说明

- 默认建议控制在 **300 字** 以内。  
- 您可以 **手动修改** 数字，告诉 AI 尽量把回答控制在该字数附近。  
- 如果 **留空**、填写 **0**、**负数** 或无法识别的内容，则视为 **不限制** 长度。  
- 字数设置是 **提示 AI 尽量控制**，实际字数可能略有出入，属正常现象。

---

## 7. 卸载方法

1. **保存文档并关闭 WPS**。  
2. 找到当初解压的安装包文件夹里的 **`uninstall.bat`**。  
3. **双击运行** `uninstall.bat`。  
4. 按提示输入 **`Y`**，按 **Enter（回车）** 确认。  
5. 等待提示 **卸载完成**。  
6. **重新打开 WPS**，检查顶部是否 **不再出现** 「源流WPS助手」。  
7. 若仍能看到入口，可 **重启电脑** 后再打开 WPS 检查一次。

卸载会删除本插件相关目录和本地配置，一般不会影响 WPS 本身及其他插件。

## 8. 常见问题

### Q1：安装后 WPS 里没有看到插件怎么办？

请依次尝试：

1. **完全退出 WPS**（任务栏不要残留 WPS 图标），再重新打开。  
2. 确认安装包是 **先解压** 再运行的 `install.bat`，而不是在压缩包内直接运行。  
3. 确认安装过程中已输入 **Y** 并回车，且窗口提示安装成功。  
4. 可右键 `install.bat`，选择 **以管理员身份运行** 后重新安装一次。  
5. 若仍无法显示，请联系您购买时的 **基础使用支持**（¥29.9 服务范围内的问题指引）。

### Q2：这个插件是否包含 DeepSeek 额度？

**不包含。**

- 您需要 **自行准备** DeepSeek API Key。  
- API 调用费用由 **您自行承担**（向 DeepSeek 官方付费/扣费）。

### Q3：DeepSeek API Key 从哪里获取？

- 请到 **DeepSeek 官方平台** 注册并申请 API Key。  
- 具体步骤以 DeepSeek 官网当前页面为准。  
- **不要** 从非官方个人处“代购 Key”，以免被骗或泄露账号。

### Q4：为什么 AI 没有回复？

常见原因包括：

- API Key **未填写** 或 **填错**、未点保存；  
- DeepSeek 账号 **余额不足**；  
- 电脑 **网络异常** 或无法访问 DeepSeek；  
- 选中文字或问题 **过长**；  
- DeepSeek 服务 **暂时繁忙或维护**。

请先检查设置里的 Key 是否已保存，并确认网络与 DeepSeek 账号状态，再根据插件里的提示排查。

### Q5：可以换电脑使用吗？

可以。本插件是 **BYOK 模式**（自带 Key 使用）：

- 在新电脑上 **重新安装** 插件；  
- 再 **填写并保存您自己的** DeepSeek API Key 即可使用。  

换机、重装等具体是否属于您的售后支持范围，以 **购买时说明** 为准。

### Q6：可以把安装包转发给别人吗？

**不建议随意转发。**

- 非官方渠道的安装包，无法保证是否被篡改、是否完整、能否获得后续说明与支持。  
- 建议让对方通过 **您购买时的官方发布渠道** 获取安装包。

### Q7：AI 生成内容可靠吗？

**不能完全依赖 AI。**

- AI 内容 **仅供参考**，可能有错漏。  
- 合同、政策、财务、对外发布等重要内容，必须 **人工核验** 后再使用。  
- **不构成** 法律、财务、投资、医疗或其他专业建议。

### Q8：如何确认已经卸载成功？

1. 重新打开 WPS 文字；  
2. 查看是否 **没有** 「源流WPS助手」入口；  
3. 若仍有显示，请 **重启电脑** 后再打开 WPS 检查一次。
